## Задание 1

```
Написать программу, которая запрашивает у пользователя натуральное число. 
Если пользователь написал неподходящее число, программа спрашивает еще раз.
Таким образом, програма должна продолжать спрашивать, пока пользователь не введет подходящее число.

Напоминание: натуральные числа это те, которые используются при подсчете (т.е. целые положительное, исключая ноль).

Дополнительное задание:
Вывести в консоль сколько попыток у пользователя заняло правильно ввести число.

Пример окончательного вывода одной программы:
    Введите натуральное число.
    -8
    Введите натуральное число.
    -11
    Введите натуральное число.
    0
    Введите натуральное число.
    4
    Вы ввели 4. Спасибо!
Пример окончательного вывода одной программы (с доп. заданием):
    Введите натуральное число.
    -1
    Введите натуральное число.
    -3
    Введите натуральное число.
    9
    Вы ввели 9. У вас получилось это сделать с 3 попытки. Спасибо!
```

## Задание 2

```
Считать из консоли начальное значение, шаг и количество элементов числовой последовательности.
В этой последовательности каждый следующий элемент должен быть ровно на один шаг больше предыдущего. 
Записать последовательность в массив, посчитать и вывести на экран сумму этих элементов.

Важно: для решения этой задачи существует математическая формула, но в данном задании нужно использовать 
именно циклы и массивы.

Пример 1:
    Ввод:
    1 
    3
    4 
    Вывод:
    Сумма = 22
```

## Задание 3 (Дополнительное)

```
Создать в программе массив с 10 случайными значениями (взять из головы). Вывести массив на экран.
Попросить пользователя выбрать одно любое значение из массива (считать из консоли значение элемента).
Вывести в консоль индекс первого элемента, значение которого равно выбранному пользователем значению. Если в массиве нет такого элемента, 
вывести соответствующее сообщение

Дополнительное задание 1:
Генерировать случайные числа с помощью генератора случайных чисел вместо того чтобы их придумывать самостоятельно

Дополнительное задание 2:
Вынести код который находит индекс значения в отдельный метод

Пример 1:
    Вывод:
    4 8 9 11 2 -6 44 5 15 32
    Введите значение любого элемента.
    Ввод:
    11
    Вывод:
    Индекс элемента с этим значением = 3
Пример 2:
    Вывод:
    4 8 9 11 2 -6 44 5 15 32
    Введите значение любого элемента.
    Ввод:
    500
    Вывод:
    В этом массиве нет такого элемента
```
